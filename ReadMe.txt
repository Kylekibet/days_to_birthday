Thanks for checking out this project!!!!!!1

Make sure you have python3 installed on your system

                            To run
                            ------

1.) Open terminal/cmd on the directory the file is.
2.) Enter python no_of_days_to_birthday.py if python3 is set by default if not:: Enter  python3 no_of_days_to_birthday.py

                             To run on linux(the above method works for all operating systems including linux)
                             ------------------------------------------------------------------------------------

1.) Right click on no_of_days_to_birthday.py file.
2.) Click on Propertise.
3.) Click on the permissions tab.
4.) 'Check mark' Allow executing file as program
5.) Open terminal on the directory the file is.
6.) enter ./no_of_days_to_birthday.py
